# Mamba (SSM) for Image Classification

## Опис проєкту

Цей проєкт присвячений дослідженню можливості використання моделі Mamba (State Space Model, SSM) для задачі класифікації зображень на датасеті CIFAR-10.
У роботі було розглянуто три підходи: використання замороженого Mamba-backbone, донавчання всієї моделі Mamba та застосування базової CNN-моделі для порівняння результатів.

## Що використовується у проєкті

У проєкті використано Python, PyTorch, torchvision, mamba-ssm, а також CUDA для прискорення обчислень на GPU.
Проєкт має модульну структуру. У папці `configs` зберігаються параметри запуску, у `data` міститься код для підготовки даних, у `models` розташовані реалізації Mamba-backbone, frozen-моделі, finetune-моделі та CNN baseline. У папці `training` знаходяться модулі для навчання та оцінювання, а в `experiments` — файли запуску експериментів.

## Короткі результати

У ході виконання проєкту було встановлено, що модель із замороженим backbone показує низьку якість класифікації та досягає приблизно 16% точності. Після донавчання всієї моделі результат покращується приблизно до 51%. Найкращий результат продемонструвала базова CNN-модель, яка досягла близько 70% точності.
Це дозволяє зробити висновок, що Mamba може бути використана для задач класифікації зображень, однак потребує донавчання, а класичні CNN-архітектури залишаються ефективнішими для CIFAR-10.

## Як розгорнути проєкт

Спочатку потрібно перейти в папку проєкту:
```bash
cd ~/projects/ssm_project
```

Потім створити та активувати віртуальне середовище:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

Після цього встановити залежності вручну:
```bash
pip install torch torchvision numpy tqdm einops
pip install causal-conv1d --no-build-isolation --no-deps
pip install mamba-ssm --no-build-isolation --no-deps
```

## Як запускати експерименти

Для запуску експерименту із замороженим backbone:
```bash
PYTHONPATH=. python experiments/run_frozen_train.py
```

Для запуску експерименту з донавчанням Mamba:
```bash
PYTHONPATH=. python experiments/run_finetune.py
```

Для запуску базової CNN-моделі:
```bash
PYTHONPATH=. python experiments/run_baseline.py
```
