## Quick Start

### Installation

#### Linux

```bash
git clone https://github.com/IPT-MMDA/ssm-orp.git
cd feat/ssm-layer-sharing
python3 -m venv .venv
source .venv/bin/activate
```

For users

```bash
pip install .
```

For developers

```bash
pip install -e .[dev]
```

***

### Training

```bash
train
```

Get help

```bash
train --help
```
