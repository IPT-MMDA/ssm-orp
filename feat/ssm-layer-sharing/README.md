### Installation

#### Linux

```bash
git clone https://github.com/IPT-MMDA/ssm-orp.git
cd feat/ssm-layer-sharing
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

### Training

```bash
python3 train.py
```
